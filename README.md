# TEDx AYZOFL YOUTH Website

TEDx AYZOFL etkinliği için hazırlanmış modern ve responsive bir website.

## 🎯 Özellikler

- 9 Konuşmacı Profili
- Detaylı Etkinlik Programı  
- Responsive Tasarım
- Smooth Animasyonlar
- Google Forms Başvuru Linki

## 🚀 Teknolojiler

- React 18+ with TypeScript
- Vite Build Tool
- CSS3 Animations
- ESLint & TypeScript Config

## 📅 Etkinlik Detayları

**Tarih:** 25 Nisan 2026  
**Mekan:** Yusuf Ziya Öner Fen Lisesi, Antalya / Döşemealtı
